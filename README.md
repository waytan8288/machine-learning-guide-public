# Machine Learning Guide — Hands-on Notebooks

Companion Colab notebooks for a self-study path through Andrew Ng's Machine Learning Specialization. Each notebook is a 30–60 minute hands-on session that implements the core algorithm of one course-week from scratch.

## Layout

```
exercises/   ← skeleton notebooks with TODOs (you fill them in)
solutions/   ← fully worked solutions with detailed explanations
```

Open the **exercises** version first. Try to fill in each TODO yourself. Open the matching **solution** only after you've genuinely tried.

## How to use

1. Click "Open in Colab" on a row above. Colab opens in your browser — no Python install required.
2. Read the markdown cells. They explain what you're doing and why.
3. Fill in each `# TODO` cell. The cell below sanity-checks your code with assertions.
4. If you get stuck, open the matching solution notebook.

## What's NOT here

- Jupyter labs from Andrew's official course (these are independent re-implementations, not transcripts)
- Boilerplate environment setup (Colab handles it)
- Production code (these are study aids, not deployable systems)
